# Travel Routes

App para buscar y mostrar rutas de viaje en un mapa interactivo.

## Stack

- **React** + Vite
- **Tailwind CSS** + shadcn/ui
- **mapcn** - Componentes de mapa basados en MapLibre GL
- **OSRM** - Routing (Open Source Routing Machine)
- **Nominatim** - Geocoding (OpenStreetMap)

## Características

- Búsqueda de rutas entre ciudades
- Visualización en mapa interactivo
- Marcadores de origen (A) y destino (B)
- Ruta dibujada en el mapa
- Información de distancia y duración

## Uso

```bash
npm install
npm run dev
```

## APIs gratuitas

- **Geocoding**: Nominatim (OpenStreetMap)
- **Routing**: OSRM
- **Mapas**: CARTO Basemaps
